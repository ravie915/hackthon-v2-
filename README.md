# 🧭 ClarityNet - AI Benefits Navigator

An advanced AI-powered platform to help people discover and access public support programs (SNAP, Medicaid, LIHEAP).

## ✨ Features

- 💬 **AI Chat Assistant** - Talk naturally about your situation
- 📊 **Visual Dashboard** - See eligibility in real-time
- 📄 **Document Manager** - Upload and scan documents
- 📅 **Appointment Scheduler** - Book with agencies
- 📈 **Application Tracker** - Monitor status
- 👤 **User Profile** - Track your journey
- 🌍 **Multi-Language** - English, Spanish, Vietnamese, Chinese, Arabic

## 📋 Requirements

- Python 3.11+
- Streamlit 1.28+
- OpenAI API Key

## 🏃 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/ravie-bit/claritynet.git
cd claritynet

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables
export OPENAI_API_KEY="your-api-key"

# 4. Run the app
streamlit run app.py
